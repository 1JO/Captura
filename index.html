Captura

Capture Screen, WebCam, Audio, Cursor, Mouse Clicks and Keystrokes

Captura is a .Net application to capture the screen along with audio from microphone and speaker output and clicks and keystrokes.

Made using C#, WPF, .Net 4.6.1


We are accepting Translations!

Now Available on Chocolatey!

choco install captura -y
